Cotizador de Criptomonedas

Single Page Aplication realizada con JavaScript y React con la cual se puede cotiza en tiempo real las más famosas criptomonedas y convertirlas en las principales monedas del mundo. Probala Acá!

E-commerce -Alura Geek-

En este proyecto/challenge propuesto por Alura Latam y Oracle para su bootcamp, había que realizar una Aplicación Web de una empresa llamada AluraGeek y debía poseer un index, una pagina de contacto, otra de login y una para agregar productos por parte del administrador. Probala Acá!

Buscador de Cocteles

Aplicación Web desarrollada con React y CSS puro, que consume una API de cocteles, en la que se puede realizar búsquedas por nombre y devuelve el coctel deseado y la información del mismo. Probala Acá!

Juego del ahorcado

Aplicación web del juego del ahorcado, es un juego de adivinanzas para dos o más jugadores. Un jugador piensa en una palabra, frase u oración y el otro trata de adivinarla según lo que sugiere por letras o dentro de un cierto número de oportunidades, realizada con HTML, CSS, JavaScript, Figma y Trello. Probala Acá!

Barbería Alura

Aplicación de una Barberia realizada con JavaScript, HTML, CSS puro y Figma. En esta aplicación se puede ver una Single Page que posee, un nav-bar, lista de precios de los distintos servicios, ubicación de la barbería con Google Maps, y más! Probala Acá!

Encriptador de textos

Aplicación que encripta textos de texto con HTML, CSS, JavaScript, Figma y Trello, para codificar texto y enviar a otra persona que posea la llave con la que fue encriptada para desencriptarlo. Probalo Acá!
